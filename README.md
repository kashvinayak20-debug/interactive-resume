# Vinayak Kashyap — Interactive Full-Stack Developer Portfolio

A dynamic, glassmorphic developer portfolio featuring:
- **Interactive Particle Canvas Background**: Mouse-reactive particle web system.
- **Movable Skill Badges**: Click and drag skill badges anywhere on screen.
- **Interactive Developer CLI Terminal**: Type live commands (`help`, `skills`, `projects`, `contact`, `theme`).
- **Live Resume Customizer**: Edit name, title, and bio live with zero page reloads.
- **Customizable Themes**: Cyberpunk Dark, Synthwave, Emerald Glass, Obsidian Midnight.
- **3D Tilt Profile Card**: Mouse-reactive perspective tilt effects.
- **PDF Resume Exporter**: 1-click PDF download formatting.

## 🚀 Quick Start

Run locally with Python:
```bash
py run.py
```
Or double-click `start.bat` on Windows to open `http://127.0.0.1:8080`.
